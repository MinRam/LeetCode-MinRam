题目

实现LRU Cache 算法

  • get(key) 返回对应值，不存在时返回-1
  • put(key,value) 存入新键值对。当key已存在时候，修改新值；
  • 要求get(key) & put(key, value) 时间复杂度为 O(1);

样例数据：

LRUCache cache = new LRUCache( 2 /* capacity */ );

cache.put(1, 1);
cache.put(2, 2);
cache.get(1);       // returns 1
cache.put(2, 3);    
cache.get(2);       // returns 3
cache.put(4, 4);    // evicts key 1
cache.get(1);       // returns -1 (not found)
cache.get(2);       // returns 3  
cache.get(4);       // returns 4

思路（逐步优化）

1、单链表实现

模拟LRU Cache机制；

struct ListNode{
    int key;
    int value;
    ListNode * next;
    ListNode(int tmpKey, int tmpValue):key(tmpKey),value(tmpValue),next(nullptr){}
};   // 链表节点结构
ListNode * head;  // 链表哨兵头节点
ListNode * tail;  // 始终指向链表的尾部

  • get(key)

    ​ 复杂度： O(n) = n

     从 <span style="color:blue">head</span> → \rightarrow → <span style="color:blue">tail</span> 进行遍历，
    • 当节点key等于所求值，返回value，并将该节点移到队尾；
    • 当找不到key, 返回value;

  • put(key, value)

    复杂度：O(n) = n

    ​ 从 head → \rightarrow → tail 进行遍历，

    • 当节点key存在时，修改value，并将该节点移到队尾；
    • 当节点key不存在时, 在链尾加入新节点(更新map,若容量超额，删除首节点head(更新map)；
2、单链表 + 哈希表实现 （内存消耗少，时间多）

模拟LRU Cache机制；

struct ListNode{
    int key;
    int value;
    ListNode * next;
    ListNode(int tmpKey, int tmpValue):key(tmpKey),value(tmpValue),next(nullptr){}
};   // 链表节点结构
ListNode * head;  // 链表哨兵头节点
ListNode * tail;  // 始终指向链表的尾部
unordered_map<int, ListNode *> map; // 存放<键，键对应节点的前一个节点>  , 无排序关系，选择unordered_map

  • get(key)

    复杂度： O(n) = 1

    从map，获取对应key节点的前一个节点 map->second;

    • 当对应key的节点不为tail，更新map节点（key的节点，以及key下一节点）信息，将该节点移动到队尾；
    • 返回 value;

  • put(key, value)

    复杂度： O(n)=1

    从map, 获取对应key节点是否存在；

    • 当节点key不存在时, 在链尾加入新节点（更新map),若容量超额，删除首节点head(更新map)，；
    • 当对应key节点存在，且不为tail，修改value,并更新map节点（key的节点，以及key下一节点）信息，将该节点移动到队尾；
3、双链表 + 哈希表实现（内存消耗多，时间少）

模拟LRU Cache机制；

struct ListNode{
    int key;
    int value;
    ListNode * pre;
    ListNode * next;
    ListNode(int tmpKey, int tmpValue):key(tmpKey),value(tmpValue),next(nullptr){}
};   // 链表节点结构
ListNode * head;  // 链表哨兵头节点
ListNode * tail;  // 始终指向链表的尾部
unordered_map<int, ListNode *> map; // 存放<键，键对应节点>  , 无排序关系，选择unordered_map

  • get(key)

    复杂度： O(n) = 1

    从map，获取对应key节点 map->second</span>;

    • 当对应key的节点不为tail，将该节点移动到队尾，返回value；
    • 当对应key的节点不存在，返回 -1;

  • put(key, value)

    复杂度： O(n)=1

    从map, 获取对应key节点是否存在；

    • 当节点key不存在时, 在链尾加入新节点（更新map),若容量超额，删除首节点head(更新map)，；
    • 当对应key节点存在，且不为tail，将该节点移动到队尾；
